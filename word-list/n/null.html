Wartość null wyraźnie wskazuje, że wartość nie istnieje i jest często używana przez programistów do wskazania pustej wartości. Wartość ta musi być wyraźnie przypisana do zmiennej, aby wskazać, że zmienna nie zawiera żadnych prawidłowych danych. 

      let value = null;
      console.log(value); // null

Operator typeof służy do określania typu danych wartości lub wyrażenia.

Typy danych można zmienić za pomocą konwersji.
