Metoda ciągów znaków trim() jest używana do usunięcia początkowych i końcowych spacji z ciągu znaków. Pozwala to "oczyścić" ciąg znaków z niepotrzebnych spacji, na przykład podczas przetwarzania wprowadzonego tekstu przez użytkownika w elementach formularza, usuwając dodatkowe spacje, które mogły zostać dodane przypadkowo. Metoda trim() nie zmienia oryginalnego ciągu znaków, lecz zwraca nowy ciąg znaków bez początkowych i końcowych spacji.

      str.trim()

      const input = "  JavaScript is awesome!  ";
      const trimmedInput = input.trim();
      console.log(trimmedInput); // "JavaScript is awesome!"
      console.log(input); // "  JavaScript is awesome!  "
