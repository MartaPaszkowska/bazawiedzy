Operator typeof służy do określania typu danych wartości lub wyrażenia. 

const quantity = 17;
log(typeof quantity); // wyświetli "number"

const message = "JavaScript is awesome!";
console.log(typeof message); // wyświetli "string"

const isSidebarOpen = false;
console.log(typeof isSidebarOpen); // wyświetli "boolean"

let username;
console.log(typeof username); // wyświetli "undefined"

let status = null;
console.log(typeof status); // wyświetli "object"

Przy null wyświeli się "object", ale de facto null nie jest objektem tylko brakiem wartości. Powodem tego jest to, że we wczesnych wersjach JavaScript, null był uważany za specjalny przypadek obiektu. Był to błąd w implementacji języka.
