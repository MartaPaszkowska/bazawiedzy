Konwersja typów (EN Type conversion) w JavaScript — to proces zmiany wartości jednego typu danych na inny typ danych. Istnieją dwa rodzaje konwersji w JavaScript: jawna i niejawna.

Jawna konwersja typów

Aby przekonwertować dowolną wartość na ciąg znaków, można użyć wbudowanej funkcji String(), która zwraca wynik jako ciąg znaków.

      console.log(String(5)); // "5"
      console.log(String(true)); // "true"
      console.log(String(false)); // "false"
      console.log(String(null)); // "null"
      console.log(String(undefined)); // "undefined"

Konwersja do Number()

      console.log(Number("5")); // 5
      console.log(Number("Molly")); // NaN
      console.log(Number(true)); // 1
      console.log(Number(false)); // 0
      console.log(Number(null)); // 0

Konwersja do Boolean()

      console.log(Boolean("wszystko oprócz poniższych daje true")); // true
      console.log(Boolean(false, 0, "", null, undefined, NaN)); // false

Konwersja do Object()

      let num = 42;
      let obj = Object(num); // [Number: 42]

Niejawna konwersja typów

Niejawna konwersja typów odbywa się automatycznie podczas wykonywania operacji lub obliczeń. Na przykład, jeśli wykonywana jest operacja dodawania (+) między ciągiem znaków a innym typem danych, JavaScript automatycznie konwertuje wartość operandu na ciąg znaków i wykonuje konkatenację ciągu znaków.

      console.log("5" + 3); // "53"
      console.log("5" + true); // "5true"

Operacje arytmetyczne (+, -, *, /) również wykonują niejawną konwersję typu.

      console.log("5" * 2); // 10
      console.log("10" - 5); // 5
      console.log(5 + true); // 6
      console.log(5 - true); // 4

Pamiętaj, że znak + obok operandów typu string powoduje ich konkatenację.

W przypadku korzystania z operatorów porównania również wykonywana jest niejawna konwersja typu. Jeśli operandy są różnych typów, są one konwertowane na liczbę przed porównaniem.

      console.log("10" > 5); // true
      console.log(10 > "5"); // true
      console.log(5 > true); // true
      console.log(5 < true); // false 
      console.log("5" < true); // false
