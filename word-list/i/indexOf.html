Metoda ciągów znaków indexOf() jest używana do wyszukiwania w ciągu znaków. Zwraca indeks pierwszego znaku podciągu, jeśli zostanie znaleziony, lub -1, jeśli podciąg nie zostanie odnaleziony. 

      str.indexOf(substr)

                       01234567|8|
      const message = "Welcome |t|o Bahamas!";
      const index = message.indexOf("to");
      console.log(index); // 8
