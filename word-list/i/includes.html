Metoda ciągów znaków includes() jest używana do sprawdzania obecności podciągu w ciągu. Zwraca wartość logiczną true, jeśli podciąg zostanie znaleziony, i false, jeśli podciąg jest nieobecny. Składnia:

      str.includes(substring)

      const username = 'Jacob Mercer';
      
      console.log(username.includes('Jacob')); // true
      console.log(username.includes('jacob')); // false
      console.log(username.includes('Mercer')); // true
      console.log(username.includes('mercer')); // false
