Klasa Math
jest wbudowaną klasą JavaScript. Klasa ta zapewnia zestaw metod do wykonywania operacji matematycznych i pracy z liczbami. Oto niektóre z najbardziej użytecznych metod dostarczanych przez klasę Math:

  • Math.floor(num) zaokrągla w dół. 
          console.log(Math.floor(1.3)); // 1
      console.log(Math.floor(1.7)); // 1
  • Math.ceil(num) zaokrągla w górę. 
          console.log(Math.ceil(1.3)); // 2
      console.log(Math.ceil(1.7)); // 2
  • Math.round(num) zwraca wartość liczby po zaokrągleniu do najbliższej liczby całkowitej. Zaokrąglenie jest wykonywane zgodnie z zasadami matematycznymi: jeśli część dziesiętna liczby jest mniejsza niż 0,5, to zaokrąglenie będzie w dół, jeśli 0,5 lub więcej, to w górę. 
          console.log(Math.round(1.3)); // 1
      console.log(Math.round(1.5)); // 2
  • Math.max(num1, num2, ...) zwraca największą liczbę ze zbioru przekazanych liczb. 
          console.log(Math.max(20, 10, 50, 40)); // 50
  • Math.min(num1, num2, ...) zwraca najmniejszą liczbę ze zbioru przekazanych liczb. 
          console.log(Math.min(20, 10, 50, 40)); // 10
  • Math.random() zwraca liczbę losową z zakresu od 0 (włącznie) do 1 (z wyłączeniem). 
          console.log(Math.random()); // liczba losowa z przedziału od 0 do 1, na przykład 0,2 ... 0,9166353649342294
