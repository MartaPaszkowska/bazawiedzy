Boolean (EN wartość logiczna) - logiczny typ danych (Boolean) ma tylko dwie wartości: true i false. Jest on używany do wykonywania operacji logicznych i sprawdzania warunków. Na przykład, na pytanie "czy światło pokoju jest włączone?" można odpowiedzieć tak (true) lub nie (false). 

      const isModalOpen = true;
      const isLoggedIn = false;

Operator typeof służy do określania typu danych wartości lub wyrażenia.

Typy danych można zmienić za pomocą konwersji.
