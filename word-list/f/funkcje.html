Funkcja to niezależny blok kodu, który wykonuje określone zadanie z różnymi wartościami początkowymi.
Deklaracja funkcji ma następującą strukturę:

  • Słowo kluczowe function.
  • Nazwa funkcji jest czasownikiem, który odpowiada na pytanie "Co zrobić?".
  • Para nawiasów ().
  • Treść funkcji w nawiasach klamrowych {}.
      function doStuff() {
      // Treść funkcji
      console.log('Log inside multiply function');
      }

Do wywołania funkcji, służy jej nazwa i para nawiasów.

      doStuff();// 'Log inside multiply function'

Parametry są zmiennymi lokalnymi, które są dostępne tylko w treści funkcji. Parametry są oddzielane przecinkami. Funkcja może mieć dowolną liczbę parametrów lub nie mieć ich wcale — w takim przypadku używane są puste nawiasy. Podczas wywoływania funkcji można przekazać argumenty w nawiasach, które są wartościami zadeklarowanych parametrów funkcji.

      // Deklaracja parametrów x, y, z
      function multiply(x, y, z) {
      console.log(`Result: ${x * y * z}`);
      }
      
      // Przekazywanie argumentów
      multiply(2, 3, 5); // "Result: 30"
      multiply(4, 8, 12); // "Result: 384"
      multiply(17, 6, 25); // "Result: 2550"

Instrukcja return służy do zwracania wartości z ciała funkcji z powrotem do kodu, do miejsca jej wywołania. Gdy interpreter napotka instrukcję return, natychmiast opuszcza funkcję (kończy jej wykonywanie) i zwraca wartość określoną po return do miejsca, w którym funkcja została wywołana. Jeśli ciało funkcji nie zawiera instrukcji return lub nie wskazuje na konkretną wartość, funkcja zwróci specjalną wartość undefined. Gdy używasz instrukcji return, wszystkie instrukcje, które następują w wierszach po niej, nie są wykonywane w ciele funkcji, ponieważ funkcja jest kończona natychmiast po napotkaniu instrukcji return.

      function multiply(x, y, z) {
      return x * y * z;
      // Zwracamy wynik wyrażenia mnożenia
      }
      
      // Wynik funkcji można zapisać w zmiennej
      const result = multiply(2, 3, 5);
      console.log(result); // 30

Gdy interpreter napotka wywołanie funkcji, zatrzymuje wykonywanie bieżącego kodu i rozpoczyna wykonywanie kodu z ciała funkcji. Po wykonaniu całego kodu w funkcji, interpreter zwraca kontrolę do miejsca, z którego funkcja została wywołana i kontynuuje wykonywanie pozostałego kodu programu.

      function multiply(x, y, z) {
      console.log(`Result: ${x * y * z}`);
      }
      
      console.log("Log before multiply execution");
      multiply(2, 3, 5); // "Result: 30"
      console.log("Log after multiply execution");

Kolejność logów w konsoli będzie następująca:

"Log before multiply execution"
"Result: 30"
"Log after multiply execution"
