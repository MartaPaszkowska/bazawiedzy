Operatory porównania są używane do porównywania dwóch wartości i zwracania wartości logicznej (true lub false) w zależności od wyniku porównania. 

      const a = 2;
      const b = 5;
      
      console.log(a > b); // false
      console.log(b > a); // true
      console.log(a >= b); // false
      console.log(b >= a); // true
