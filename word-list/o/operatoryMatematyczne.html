JavaScript posiada wiele operatorów matematycznych, które umożliwiają wykonywanie obliczeń arytmetycznych. 

      const x = 8;
      const y = 5;
      console.log(x + y); // 13// zwraca wynik dodawania
      console.log(x - y); // 3 // zwraca wynik odejmowania
      console.log(x * y); // 40 // zwraca wynik mnożenia
      console.log(x / y); // 1.6 // zwraca wynik dzielenia
      console.log(x % y); // 3 // zwraca resztę z dzielenia
      console.log(x ** y);// 32768 // potęguje jedną liczbe przez drugą

Dla wszystkich operatorów arytmetycznych istnieją równoważne operatory złożone. Pozwalają one wykonywać operacje arytmetyczne w bardziej kompaktowy sposób.

Operatory arytmetyczne i ich złożone odpowiedniki:
Dodawanie: +=. Przykład: x += y jest równoważne x = x + y.
Odejmowanie: -=. Przykład: x -= y jest równoważne x = x - y.
Mnożenie: *=. Przykład: x * = y jest równoważne x = x * y.
Dzielenie: /=. Przykład: x /= y jest równoważne x = x / y.
Reszta z dzielenia: %=. Przykład: x %= y jest równoważne x = x % y. 


      let age = 25;
      age += 1; czyli age + 1 = age;
      console.log(age); // 26

UWAGA Podczas wykonywania operacji z liczbami ułamkowymi mogą wystąpić niedokładności spowodowane wewnętrzną reprezentacją liczb w pamięci komputera.

      console.log(0.1 + 0.2 === 0.3); // false
      console.log(0.1 + 0.2); // 0.30000000000000004

Są na to dwa sposoby:

      console.log((0.1 * 10 + 0.2 * 10) / 10); // 0.3

lub za pomocą metody toFixed(). W celu zaokrąglenia wyniku do określonej liczby miejsc po przecinku. Metoda ta zwraca string więc jeżeli chcemy otrzymać liczbę należy ją przekonwerować

      console.log((0.1 + 0.2).toFixed(1)); // '0.3'
      console.log((5).toFixed(2)); // '5.00'
      console.log((8.762195).toFixed(4)); // '8.7622'
