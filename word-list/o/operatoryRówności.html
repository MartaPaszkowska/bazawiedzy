JavaScript posiada operatory równości, które pozwalają porównywać wartości i określać, czy są one równe, czy nie.

Operatory ścisłej równości:
Operator === (ścisła równość) — porównuje dwie wartości pod kątem równości, zwraca true, jeśli wartości operandów są równe i mają ten sam typ danych. W przeciwnym razie zwraca false.
Operator !== (ścisła nierówność) — porównuje dwie wartości pod kątem nierówności, zwraca true, jeśli wartości operandów nie są równe lub mają różne typy danych. W przeciwnym razie, zwraca false. 

      console.log(5 === 5); // true
      console.log(5 === "5"); // false
      console.log(5 !== "5"); // true
      console.log(5 !== 5); // false
      console.log(1 === true); // false
      console.log(1 !== true); // true
