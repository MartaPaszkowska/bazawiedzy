Operator trójargumentowy zaleca się stosować w najprostszych przypadkach operacji przypisania lub zwracania gdzie odpowiedz to tak lub nie. Budowa:

      <condition> ? <expression if condition is true> : <expression if condition is false>

Przykładowy różnica w zastosowaniu opratora trójargumentowego i instrukcji if:

      let type;
      const age = 20;
      
      if (age >= 18) {
      type = 'adult';
      } else {
      type = 'child';
      }
      
      console.log(type); // 'adult'
      

      const age = 20;
      const type = age >= 18 ? 'adult' : 'child';
      console.log(type); // 'adult'
