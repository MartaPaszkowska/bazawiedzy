Zmienna (EN variable) to kontener używany do przechowywania danych.

Deklaracja zmiennej zaczyna się od słowa kluczowego const lub let. Główną różnicą między const i let jest możliwość nadpisania wartości zmiennej po jej zadeklarowaniu.

Słowo kluczowe const zapobiega nadpisywaniu wartości zmiennej.
Słowo kluczowe let pozwala na zmianę wartości zmiennej po jej zadeklarowaniu. Dla zmiennych zadeklarowanych za pomocą let nie jest konieczne natychmiastowe ustawienie wartości.

Nazwę zmiennej zapisujemy w notacji camelCase. Warto pamiętać, że istnieją zarezerwowane słowa kluczowe. 

    const age = 20; //deklaracja zmiennej const

    console.log(username); //"Mango" // wywołanie zmiennej w konsoli 

    let username; //deklaracja zmiennej let bez ustawienia wartości początkowej

    console.log(username); //undefined // wywołanie zmiennej bez ustawienia wartości początkowej

    username = "Poly"; // nadpisanie wartości

    console.log(username); //"Poly" // wywołanie zmiennej po nadpisaniu wartości


let username = "Mango"; //deklaracja zmiennej let
