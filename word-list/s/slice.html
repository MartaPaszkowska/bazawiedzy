Metody slice() jest używana do tworzenia kopii części lub całego ciągu znaków bez zmiany oryginalnego ciągu. Trzeba tylko określić indeks początkowy i końcowy. Składnia metody slice() wygląda następująco:

      str.slice(startIndex, endIndex)

      const fullName = "Jacob Mercer";
      console.log(fullName.slice(0, 4)); // 'Jaco'
      console.log(fullName.slice(3, 9)); // 'ob Mer'
      console.log(fullName.slice(0, fullName.length)); // 'Jacob Mercer'

Parametr endIndex jest opcjonalny. Jeśli endIndex nie jest określony, wyodrębniane są wszystkie elementy aż do końca ciągu.

      const fullName = "Jacob Mercer";
      console.log(fullName.slice(1)); // 'acob Mercer'
      console.log(fullName.slice(3)); // 'ob Mercer'

Jeśli wywoła się metodę slice() bez argumentów, tworzy ona dokładną kopię ciągu i zwraca ją.

      const fullName = "Jacob Mercer";
      console.log(fullName.slice()); // 'Jacob Mercer'
