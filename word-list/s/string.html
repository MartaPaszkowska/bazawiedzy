Ciąg znaków (String) — to sekwencja znaków ujęta w pojedyncze ' ' lub podwójne "" cudzysłowy. 

      const username ='Mango995';
      const description ="JavaScript is awesome!";

Jeśli zastosujesz operator + do ciągu znaków i dowolnego innego typu danych, wynikiem będzie nowy ciąg znaków zawierający połączenie oryginalnych wartości. Nazywa się to konkatenacją.

      const message = "Mango " + "is" + " happy";
      console.log(message); // "Mango is happy"

      const age = 24;
      const message = "Poly is " + age + " years old!";
      console.log(message); // "Poly is 24 years old!"

      console.log("Mango" + true); // "Mangotrue"

      console.log(1 + 2 + "4"); // "34"

Każdy znak ma swój własny numer sekwencji (indeks). Indeksowanie elementów ciągu rozpoczyna się od zera. Pierwszy znak ma indeks 0, drugi ma indeks 1, trzeci ma indeks 2 itd.

Aby uzyskać dostęp do określonego znaku w ciągu znaków, używamy składni nawiasów kwadratowych, w których określamy indeks żądanego znaku:

      const product = "Repair droid";
      console.log(product[0]); // 'R'
      console.log(product[5]); // 'r'

Aby znaleźć indeks ostatniego znaku ciągu, odejmij jeden od długości tego ciągu string.length - 1

      const product = "Repair droid";
      const lastElementIndex = product.length - 1;
      console.log(product[lastElementIndex]); // 'd'

      const product = "Repair droid";
      console.log(product[product.length - 1]); // 'd'

Operator typeof służy do określania typu danych wartości lub wyrażenia.
